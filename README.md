# QVAR: A Quantum Variational Regularization Method for Linear Inverse Problems

This repository contains the code and archived experimental results used in the manuscript

"QVAR: A Quantum Variational Regularization Method for Linear Inverse Problems".

## Authors

Siiri Rautio
Hjördis Schluter
Andreas Hauptmann
Babak Maboudi Afkham

## Repository structure

- simulator/: quantum simulation codes
- hardware/: quantum hardware execution codes
- results/: archived measurement results
- figures/: scripts used to generate manuscript figures

## Reproducibility

The repository contains the source code and data required to reproduce the numerical experiments presented in the manuscript.
